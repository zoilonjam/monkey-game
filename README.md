Feed the Monkey 🐵🍌
Description

Feed the Monkey is a fun and interactive browser-based game where  a player help a hungry monkey catch falling bananas while avoiding dangerous obstacles. The objective is to complete the missions  before running out of lives.

The game is designed to improve hand-eye coordination, reaction time, and player engagement through simple yet challenging gameplay mechanics.

Features
🍌 Catch falling bananas to earn points
🐵 Control a monkey character using keyboard or touch controls
❤️ Life system that decreases when rocks are hit
🎯 Score tracking
🌿 Animated jungle-themed environment
🍃Leaves Fluttering in Wind 
☁️ Moving clouds,eagle,butterfly and environmental effects
📱 Mobile-friendly responsive design
🔊 Sound effects and visual feedback
How to Play
Desktop Controls
Left Arrow (←) – Move monkey left
Right Arrow (→) – Move monkey right
Mobile Controls
Tap and hold the left  arrow  to move left
Tap and hold the right arrow of the screen to move right
Objective
Catch as many bananas as possible.
Avoid obstacles such as falling rocks.
Earn points for every banana collected.
Survive as long as possible and achieve the highest score.
